<h1 align="center"> 
  $\textnormal{\color{purple}{INSTALLATION}}$
</h1>

1. Clone the repository:
   ```diff
   git clone https://github.com/ridika-2004/Hydration-Tracker.git
   ```

2. Compile the Java files:
   ```diff
   javac -d bin src/**/*.java
   ```

3. Run the application:
   ```diff
   java -cp bin App.App
   ```

<br><br>
